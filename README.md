# How to dump yourself

1. Clone [Dumper-7](https://github.com/Encryqed/Dumper-7)
2. Build it
3. Run Half Sword
4. Inject Dumper-7 into Half Sword
5. Wait for the dump to complete
6. The dump will be saved to `C:/Dumper-7/<game version>`
